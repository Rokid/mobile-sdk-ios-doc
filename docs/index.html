Rokid Mobile SDK

介绍

Rokid Mobile SDK 主要 为企业开发者 提供整套的 移动开发支持，能够为 开发者的移动应用拥有 与 Rokid设备之间 配网、指令发送、技能等交互能力。

本文档提供了 Rokid Mobile SDK iOS 版的集成方式、API解释、Demo 代码等。
